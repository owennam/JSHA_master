## Common Section Templates

### Hero Section Variations

#### Version 1: Centered Text
```
Large headline
Supporting subheadline
Primary CTA button
Secondary CTA button (optional)
Hero image or illustration
```

#### Version 2: Split Layout
```
Left: Text content + CTAs
Right: Hero image or video
```

#### Version 3: Full-Screen Background
```
Background image (dark overlay)
Centered white text
Single prominent CTA
Scroll indicator
```

### Feature Section Variations

#### Version 1: Icon + Title + Description
```
Grid of 3-6 cards
Each with: Icon, Title, Description
Hover effects
```

#### Version 2: Alternating Image/Text
```
Image on left, text on right
Next row: text on left, image on right
Repeat pattern
```

#### Version 3: Timeline
```
Vertical timeline
Each step with number/icon
Title and description
Progressive disclosure
```

### Testimonial Section Variations

#### Version 1: Card Grid
```
Grid of testimonial cards
Each with: Quote, Name, Title, Photo
Star ratings
Company logos below
```

#### Version 2: Carousel
```
Large centered testimonial
Navigation arrows
Dots for pagination
Auto-rotate option
```

#### Version 3: Video Testimonials
```
Video thumbnail with play button
Quote below video
Name and credentials
Grid or carousel layout
```

## Content Writing Tips by Industry

### SaaS
- Focus on problem-solving
- Use action-oriented language
- Highlight time/cost savings
- Include free trial offers
- Show product screenshots

### Professional Services
- Emphasize expertise and credentials
- Use testimonials and case studies
- Focus on outcomes and results
- Include professional headshots
- Clear contact/booking process

### E-commerce
- Product benefits over features
- Use urgency and scarcity
- Show reviews and ratings
- Clear pricing and shipping
- Multiple product images

### Restaurant/Food
- Mouthwatering descriptions
- High-quality food photography
- Emphasize ambiance and experience
- Clear menu and pricing
- Easy reservation process

### Nonprofit
- Emotional storytelling
- Show impact with statistics
- Make donation process simple
- Highlight transparency
- Include volunteer opportunities

## Industry-Specific Design Patterns

### Tech/SaaS
- Clean, modern design
- Lots of whitespace
- Bold typography
- Blue/purple color schemes
- Dashboard/product screenshots

### Creative/Agency
- Bold, unique designs
- Large portfolio images
- Animation and interactions
- Dark or high-contrast themes
- Showcase personality

### Professional Services
- Conservative, trustworthy design
- Professional photography
- Blue/gray color schemes
- Clear information hierarchy
- Credibility indicators

### Retail/E-commerce
- Product-focused design
- Large, zoomable images
- Clear navigation
- Prominent CTAs
- Trust badges

### Restaurant/Food
- Warm, inviting colors
- Appetizing photography
- Easy-to-read menus
- Location prominence
- Reservation/order CTAs
